# **App Name**: Token Miner

## Core Features:

- User Authentication: Allow users to sign up with their full name, email, mobile number, gender, password, and confirm password. Implement login functionality and password reset.
- Homepage Access: Allow users to access the homepage without logging in.
- Mining Restriction: Require users to log in or sign up to start mining.
- Sidebar Navigation: Implement a sidebar with login/signup options if the user is not logged in. Replace these with a logout option and profile section upon login.
- Persistent Login: Maintain user login state until they explicitly log out.
- Profile Management: Provide a profile section in the sidebar with an option to edit profile information.
- Mining Rate Display: Display the mining rate (1 Param coin per hour) on the screen.
- Cloud-Based Coin Saving: Save user's mined coins in the cloud to prevent data loss.
- Mining Simulation: Simulate the mining process and update the token count in real-time. Display the current amount of mined Param tokens.

## Style Guidelines:

- Primary color: Dark teal (#008080) for a professional, trustworthy feel. Teal communicates both growth and stability, fitting the app's purpose.
- Background color: Dark gray (#222222) to provide contrast and modern feel.
- Accent color: Lime green (#32CD32) for highlights and active elements. This hue is analogous to the teal, but significantly brighter.
- Body and headline font: 'Inter' sans-serif, providing a clean and modern look that is highly readable.
- Use minimalist icons for key actions and information display.
- Use a clean, grid-based layout to ensure content is well-organized and easy to navigate.
- Incorporate subtle animations for feedback during token mining, and other status updates.
